## Potentially Similar Patterns to Investigate

Any module that:
1. Uses OnServerEvent to receive parameters
2. Passes those parameters directly to applyStatus duration
3. Uses client parameters in task.wait/task.delay

Grep pattern to find similar issues:
```
applyStatus\([^,]+,\s*"[^"]+",\s*arg[0-9]
```
